## 시작하기

### 1️⃣ 로그인

**주소**: http://localhost:3000 (또는 배포된 URL)

**로그인 단계**:

```
1. 이메일 입력
   📧 예: therapist@akms.local

2. 비밀번호 입력
   🔐 예: password123

3. 역할 선택 (드롭다운)
   👤 admin
   👨‍⚕️ therapist
   👨‍👩‍👧 parent

4. "로그인" 버튼 클릭
```

**테스트 계정**:

| 역할 | 이메일 | 비밀번호 | 접근 페이지 |
|------|--------|----------|-----------|
| 관리자 | admin@akms.local | password123 | 모든 페이지 |
| 치료사 | therapist@akms.local | password123 | 대시보드, 스케줄, 분석 |
| 부모 | parent@akms.local | password123 | 자녀 성장, 메시지 |

## 관리자(Admin) 완전 가이드

### 📊 대시보드 (`/dashboard`)

**목적**: 전체 시스템 개요, 오늘의 일정, 통계

**주요 요소**:

#### 1. 오늘의 스냅샷 (상단)
```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│ 총 아동 수  │ 활성 세션   │ 완료율      │ 평균 점수    │
│     4명     │    6개      │   78.5%     │   82.3점    │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

**조작 방법**:
- 각 카드 클릭 → 상세 보기로 이동
- 날짜 범위 선택 → 통계 업데이트 (좌측 필터)

#### 2. 주간 달력 (중앙)
- 월-토요일 시간표
- 각 세션별 아동 색상 표시
- 시간 충돌 자동 감지 (⚠️ 빨간색 표시)

**조작 방법**:
- 세션 클릭 → 상세 정보 팝업
- 빈 칸 더블클릭 → 새 세션 추가
- 세션 드래그 → 시간 변경

#### 3. 이번 주 성과 (하단)
- 완료된 세션 목록
- 평균 학습 곡선 그래프
- 상위 성과 아동 목록

### 👶 아동정보 관리 (`/children`)

**목적**: 모든 아동의 정보 조회, 추가, 수정, 삭제

**화면 구조**:

```
┌─────────────────────────────────────────┐
│  👶 아동정보 관리                        │
│  [+ 아동 추가] [🔍 검색] [필터 ▼]      │
├─────────────────────────────────────────┤
│  ┌──────┐  ┌──────┐  ┌──────┐          │
│  │ 민준 │  │ 소영 │  │ 지호 │  ...    │
│  │ 3세  │  │ 3세  │  │ 4세  │          │
│  │ 카드 │  │ 카드 │  │ 카드 │          │
│  └──────┘  └──────┘  └──────┘          │
│                                         │
│  (반응형 그리드: Desktop 4열, Tablet 2열, Mobile 1열)
│                                         │
└─────────────────────────────────────────┘
```

#### 아동 추가 (`+ 아동 추가` 버튼)

```
1. 모달 열기
   ↓
2. 필수 정보 입력
   ├─ 이름: 민준
   ├─ 생년월일: 2021-01-15
   ├─ 나이: (자동 계산)
   ├─ 전화: 010-1234-5678
   ├─ 주소: 서울시 강남구...
   └─ 색상: (8가지 파스텔 선택)
   ↓
3. 사진 첨부 (선택)
   ├─ 드래그 & 드롭
   └─ 또는 파일 선택
   ↓
4. 메모 추가 (선택)
   └─ 특이사항, 관심사 등
   ↓
5. "저장" 버튼
```

#### 아동 정보 수정

```
1. 카드에서 [✏️ 수정] 클릭
   ↓
2. 수정 모달 열기
   ├─ 모든 필드 수정 가능
   └─ 사진 업데이트 가능
   ↓
3. "저장" 버튼
```

#### 아동 정보 삭제

```
⚠️ 주의: 삭제할 수 없음 (데이터 무결성)
→ 대신 "비활성화" 기능 사용
  1. 카드에서 [⋯] 메뉴
  2. "비활성화" 선택
  3. 확인
```

### 📅 스케줄 관리 (`/schedule`)

**목적**: 주간 시간표 작성 및 관리

**시간대**:
- 08:00-09:00 (1차)
- 10:00-11:00 (2차)
- 14:00-15:00 (3차)
- 16:00-17:00 (4차)

**세션 추가**:

```
1. 빈 칸 더블클릭
   ↓
2. 세션 추가 모달
   ├─ 요일: 월요일 (자동)
   ├─ 시간대: 08:00-09:00 (자동)
   ├─ 아동: 민준 (선택)
   ├─ 세션명: 한글 학습 (입력)
   └─ 색상: 민준의 색상 (자동)
   ↓
3. "저장" 버튼
```

**세션 편집**:

```
1. 세션 클릭 → "편집" 버튼
   ↓
2. 수정 모달
   ├─ 아동 변경 가능
   ├─ 세션명 변경
   └─ 색상 변경 (선택)
   ↓
3. "저장" 버튼
```

**주의사항**:
- ⚠️ 시간 충돌 감지 → 빨간색 경고
- 같은 아동을 같은 시간에 2번 이상 배치 불가
- 주말은 토요일만 사용 가능

### 📈 분석 페이지 (4가지)

#### 1️⃣ ABC 행동 분석 (`/abc-analysis`)

**목적**: 아동의 행동 패턴 분석

```
상단: 아동 선택
└─ 드롭다운에서 민준, 소영, 지호, 연서 선택

중앙: 패턴 테이블
┌─────────────────────────────────────────┐
│ 선행조건(A) | 행동(B) | 결과(C) | 빈도  │
├─────────────────────────────────────────┤
│ 새 장난감  │ 침착할 수 없음 │ 격리 │ 3회 │
│ 음식 거부  │ 지랄 떨기  │ 간식 제공 │ 5회 │
└─────────────────────────────────────────┘

하단: 그래프
└─ 패턴별 빈도 막대 그래프
```

**사용 방법**:
1. 아동 선택
2. 날짜 범위 설정 (기본: 최근 7일)
3. 분석 결과 자동 표시
4. 클릭하여 상세 보기

---

#### 2️⃣ 행동 예측 모델 (`/behavior-prediction`)

**목적**: 미래 행동 예측

```
입력: 최근 세션 로그
↓
분석: 패턴 추출 + 머신러닝 알고리즘
↓
출력: 예측 행동 + 신뢰도
```

**화면**:
```
상단: 아동 선택 + 예측 모델 선택
├─ Model A (정확도: 85%)
├─ Model B (정확도: 78%)
└─ Model C (정확도: 72%)

중앙: 예측 결과
├─ 예측 행동: "침착할 수 없음" (확률: 82%)
├─ 권장 중재: "감각 장난감 제공"
└─ 신뢰도: ████████░ 80%

하단: 예측 정확도 추이 그래프
```

---

#### 3️⃣ 중재 효과 분석 (`/intervention-analysis`)

**목적**: 특정 중재의 효과 측정

```
1. 중재 선택
   └─ 예: "타임아웃", "보상시스템", "감각장난감"

2. 아동 선택
   └─ 민준

3. 기간 선택
   └─ 2026-04-01 ~ 2026-04-30

4. 분석 결과 자동 계산
   ├─ 적용 횟수: 15회
   ├─ 성공률: 73%
   ├─ 평균 효과: 85점/100점
   └─ 트렌드: 상승 ↑
```

---

#### 4️⃣ 학습 속도 추적 (`/learning-velocity`)

**목적**: 아동의 학습 진도 속도 측정

```
Y축: 점수 (0-100)
X축: 날짜 (최근 30일)

그래프:
100 |      ╱ ╱
 75 | ╱ ╱ ╱ ╱
 50 |╱ ╱ ╱
 25 |
  0 └──────────────
    1일 7일 14일 21일 30일

좌측: 통계
├─ 초기 점수: 45점
├─ 현재 점수: 85점
├─ 진행률: +89%
├─ 평균 증가: 1.3점/일
└─ 예상 목표 달성일: 2026-05-15
```

### 📅 일일 세션 기록 (`/schedule`)

**가장 중요한 기능**

#### Step 1: 세션 선택

```
주간 시간표에서 해당 세션 클릭
↓
"세션 시작" 버튼
```

#### Step 2: 과제 선택

```
모달 열기:
┌──────────────────────────────┐
│  세션 정보                    │
├──────────────────────────────┤
│ 아동: 민준                   │
│ 날짜: 2026-04-27 (월)        │
│ 시간: 08:00-09:00           │
│                              │
│ 발달영역 선택:               │
│ ├─ 읽기 ▼                    │
│                              │
│ 장기목표(LTO) 선택:          │
│ ├─ LTO 1: 한글 인식 ▼        │
│                              │
│ 단기목표(STO) 선택:          │
│ ├─ STO 1-1: ㄱ 인식 ▼        │
│ ├─ STO 1-2: ㄴ 인식          │
│ └─ STO 1-3: 단어 읽기        │
└──────────────────────────────┘
```

#### Step 3: 점수 입력

```
점수 입력 (0-100):
┌─────────────────────┐
│ [========================] 85점 │
│ 0    25    50    75   100 │
└─────────────────────┘

또는 직접 입력: [85] 점
```

**점수 해석**:
- 90-100: 완벽함 (마스터)
- 70-89: 좋음 (능숙함)
- 50-69: 보통 (학습 중)
- 30-49: 부족함 (추가 연습 필요)
- 0-29: 아직 미달성

#### Step 4: 관찰 메모 추가

```
행동 관찰 필드:
┌───────────────────────────────┐
│ ABC 분석 (선택):              │
│                               │
│ 선행조건(A):                   │
│ □ 새로운 과제 제시            │
│ □ 친구들과 상호작용           │
│ □ 기타: ____________          │
│                               │
│ 행동(B):                      │
│ □ 집중력 부족                 │
│ □ 적극적 참여                 │
│ □ 기타: ____________          │
│                               │
│ 결과(C):                      │
│ □ 긍정적 피드백              │
│ □ 보상 제공                   │
│ □ 기타: ____________          │
│                               │
│ 추가 메모:                    │
│ [오늘 매우 활발했고 집중력이 좋았음...]
└───────────────────────────────┘
```

#### Step 5: 저장 및 자동 처리

```
"저장" 버튼 클릭
↓
자동으로:
├─ ABC 분석 패턴 추출
├─ 학습 속도 계산
├─ 협업 대시보드 업데이트
├─ 부모에게 알림 발송 (선택)
└─ 차트 업데이트
```

## 자주 묻는 질문(FAQ)

### 🔐 계정 & 로그인

**Q1: 비밀번호를 잊어버렸어요**
```
A: 로그인 페이지의 "비밀번호 찾기" 링크 클릭
   → 이메일 입력
   → 인증 메일 수신 (5분 이내)
   → 링크 클릭하여 새 비밀번호 설정
```

**Q2: 계정이 잠겼어요**
```
A: 5회 실패 후 15분 동안 잠김
   → 15분 후 자동 해제
   → 또는 관리자에게 연락
```

**Q3: 여러 기기에서 로그인할 수 있나요?**
```
A: 네, 가능합니다.
   → 각 기기에서 독립적으로 로그인 가능
   → 데이터는 실시간 동기화됨
```

---

### 📊 데이터 & 기록

**Q4: 세션 로그를 수정할 수 있나요?**
```
A: 예, 가능합니다.
   Step 1: Schedule 페이지 → 세션 선택
   Step 2: [수정] 버튼 클릭
   Step 3: 정보 변경
   Step 4: [저장] 버튼

   주의: 관리자는 이미 저장된 기록도 수정 가능
        치료사는 자신이 입력한 기록만 수정 가능
```

**Q5: 이전 데이터를 내보낼 수 있나요?**
```
A: 예, 관리자만 가능합니다.
   Step 1: Dashboard → [데이터 내보내기]
   Step 2: 기간 선택
   Step 3: 형식 선택 (CSV, Excel, PDF)
   Step 4: [다운로드] 클릭
```

**Q6: 실수로 세션을 삭제했어요**
```
A: 삭제 후 12시간 이내 복구 가능
   Step 1: Dashboard → [휴지통]
   Step 2: 해당 세션 선택
   Step 3: [복구] 버튼
   
   주의: 12시간 후 자동 영구 삭제
```

## 문제 해결 가이드

### 🔴 심각한 문제

#### 1. 로그인 불가
```
증상: "이메일 또는 비밀번호가 잘못되었습니다" 계속 나옴

진단:
☐ 이메일 오타 확인
☐ Caps Lock 확인
☐ 비밀번호 복사-붙여넣기 (공백 제거)

해결:
Step 1: 비밀번호 초기화
   → "비밀번호 찾기" 링크
   → 이메일 입력
   → 인증 메일 확인
   → 새 비밀번호 설정
   
Step 2: 여전히 안되면 관리자 연락
```

#### 2. 페이지 로딩 실패
```
증상: 페이지가 계속 로딩 중...

진단:
☐ 인터넷 연결 확인
☐ 브라우저 콘솔 (F12) 에러 확인
☐ 다른 탭에서 로그인 상태 확인

해결:
Step 1: 페이지 새로고침 (Ctrl + Shift + R)
Step 2: 브라우저 캐시 삭제
Step 3: 다른 브라우저 시도
Step 4: 관리자에게 로그 파일 함께 보내기
```

#### 3. 데이터 손실
```
증상: 저장한 데이터가 사라졌어요

진단:
☐ 저장 버튼을 실제로 클릭했는지 확인
☐ 네트워크 오류 없었는지 확인
☐ 다른 사용자가 동시 수정했는지 확인

해결:
Step 1: 휴지통 확인 (Dashboard → [휴지통])
Step 2: 복구 버튼 클릭 (12시간 이내)
Step 3: 복구 불가면 관리자 연락 (DB 백업 복구)
```

---

### 🟡 중간 정도 문제

#### 4. 점수 입력 후 저장 안됨
```
증상: "저장" 클릭했는데 계속 로딩...

진단:
☐ 필수 필드 모두 입력했는지 확인
   ├─ 아동 선택
   ├─ 발달영역 선택
   ├─ LTO 선택
   ├─ STO 선택
   └─ 점수 입력 (0-100)
☐ 네트워크 연결 확인

해결:
Step 1: [저장] 다시 클릭
Step 2: 2-3초 대기
Step 3: 모달 닫고 다시 오기
Step 4: 다른 기기에서 시도
```

#### 5. 메시지 전송 실패
```
증상: 메시지가 보내지지 않음

진단:
☐ 상대방이 로그인했는지 확인
☐ 인터넷 연결 확인
☐ 메시지 길이 확인 (1000자 제한)

해결:
Step 1: 새로고침
Step 2: 메시지 다시 작성
Step 3: 첨부파일 제거하고 다시
Step 4: 브라우저 다시 시작
```

#### 6. 그래프가 표시 안됨
```
증상: 빈 차트 또는 "데이터 없음"

진단:
☐ 세션 로그가 존재하는지 확인
   → Schedule 페이지에서 기록 확인
☐ 날짜 범위 확인
   → 7일 이상의 데이터 필요

해결:
Step 1: 세션 로그 몇 개 추가
Step 2: 2-3분 대기 (분석 시간)
Step 3: 페이지 새로고침
Step 4: 날짜 범위 확장 (예: 30일)
```

### 🟢 경미한 문제

#### 7. 색상이 이상해요
```
증상: 아동 색상이 다르게 표시됨

원인: 브라우저 캐시

해결:
Step 1: Ctrl + Shift + Delete
Step 2: "캐시된 이미지와 파일" 선택
Step 3: 시간 범위: "모든 시간"
Step 4: "데이터 삭제"
Step 5: 페이지 새로고침
```

#### 8. 필드 레이블이 겹쳐요
```
증상: 작은 화면에서 텍스트 겹침

원인: 브라우저 줌 레벨

해결:
Step 1: Ctrl + 0 (줌 리셋)
Step 2: 또는 Ctrl + - (축소)
```

#### 9. 알림이 너무 많아요
```
증상: 너무 많은 알림 받음

해결:
Step 1: Settings → [알림]
Step 2: 알림 타입별 체크박스 조정
   ├─ ☑ 새 메시지
   ├─ ☐ 완료된 세션 (미체크)
   ├─ ☑ 마일스톤
   └─ ☐ 일일 리포트 (미체크)
Step 3: [저장]
```
