## 📋 사용자 요청 사항별 검증

### 1️⃣ 스케줄 관리 (Schedule)

#### 요청사항
- [ ] 월-토 시간표 형식 (전체 화면에 한 번에 표시)
- [ ] 요일을 중복으로 선택 가능
- [ ] 오전/오후 분할
- [ ] 아동별 색상 구분
- [ ] 아동명, 세션, 시작시간, 끝시간 입력
- [ ] 중복 선택하여 수정/삭제/저장 가능

#### 현재 구현 상태
- ❌ **아직 미구현** - Schedule.tsx 파일에 실제 시간표 기능이 없음
- 필요: 시간표 UI 컴포넌트 + Context 연동

#### 검증 항목
```
□ Schedule.tsx에서 시간표 형식의 테이블 렌더링
□ 월-토 요일 표시 + 오전(09:00-12:00) / 오후(13:00-18:00) 분할
□ 아동 선택 시 색상 변경
□ 중복 선택 기능 (Multi-select) 구현
□ 저장/수정/삭제 버튼 동작 확인
□ 데이터 Context에 저장 확인
```

### 3️⃣ 데이터 기록지 (SessionLog)

#### 요청사항
- [ ] 아동별로 열기 가능 (아동명 클릭 → 데이터 표시)
- [ ] 과제들을 카드형으로 나열
- [ ] 과제별 그래프 버튼 (클릭 시 해당 과제의 그래프 표시)
- [ ] 수정/삭제/저장 버튼
- [ ] 비고란 (여러 줄 입력)
- [ ] 일일 데이터 작성 가능

#### 현재 구현 상태
- ✅ **구현 완료** - 커리큘럼과 연결된 드롭다운으로 과제 선택
- ✅ SessionLog.tsx에 아동 선택, 날짜 선택 기능
- ✅ 과제 카드형 렌더링
- ✅ 수정/삭제/저장/완료 버튼
- ❌ 그래프 버튼 아직 기능 미구현

#### 검증 항목
```
✅ 아동 선택 드롭다운
✅ 날짜 선택 (일일 단위)
✅ 커리큘럼 드롭다운 (발달영역 → LTO → STO)
✅ 과제 추가 기능
✅ 시간 입력 필드 (시작시간, 종료시간)
✅ 성과 점수 슬라이더
✅ 비고 텍스트 에어리어
✅ 수정/삭제/저장 버튼
✅ 완료 버튼 (완료목록으로 이동)
⚠️ 그래프 버튼 (아직 기능 없음 - 차후 구현)
```

**다음 구현 필요**: 그래프 버튼 클릭 시 모달이나 팝업으로 차트 표시

## 🔧 기술 구현 상세

### Context API 통합
```typescript
CurriculumContext
├─ domains: DevelopmentDomain[]
├─ sessionTasks: SessionTask[]
├─ completionTasks: SessionTask[]
└─ 관리 함수들
  ├─ Domain CRUD (addDomain, editDomain, deleteDomain)
  ├─ LTO CRUD (addLTO, editLTO, deleteLTO)
  ├─ STO CRUD (addSTO, editSTO, deleteSTO)
  └─ Task 관리 (add, update, delete, complete)
```

### 페이지 구조
```
App.tsx
├─ CurriculumProvider (전역 상태 관리)
│  └─ CacheProvider (오프라인 캐시)

│        ├─ Dashboard
│        ├─ Schedule (⚠️ 아직 시간표 구현 필요)
│        ├─ Children (✅ UI 있음, 저장 기능 강화 필요)
│        ├─ SessionLog (✅ 완성)
│        ├─ Completion (✅ 완성)

```

## 🧪 수동 테스트 체크리스트

### 커리큘럼 관리 테스트
```
□ 커리큘럼 페이지 열기
□ 발달영역 추가 (예: "사회성발달")
□ LTO 추가 (예: "사회 상호작용")
□ STO 추가 (예: "눈맞춤 유지")
□ 각각 수정/삭제 동작 확인
□ 브라우저 새로고침 후 데이터 유지 확인 (indexedDB)
```

### 데이터 기록지 테스트
```
□ 아동 선택 (예: "민준")
□ 날짜 선택
□ 과제 추가 (발달영역 선택)
□ LTO 자동 필터링 확인
□ STO 자동 필터링 확인
□ 시간, 점수, 비고 입력
□ 수정 버튼 클릭 → 수정 가능 확인
□ 저장 버튼 클릭 → 데이터 저장 확인
□ 완료 버튼 클릭 → 완료목록으로 이동 확인
□ 삭제 버튼 클릭 → 과제 삭제 확인
```

### 완료목록 테스트
```
□ 완료목록 페이지 열기
□ 새로운 완료 과제 표시 확인
□ 아동별 필터링 동작
□ 시간 범위 필터링 동작
□ 정렬 (최신순/오래된순) 동작
□ 통계 (완료수, 평균 점수) 정확도
□ Excel 내보내기 (다운로드 확인)
```

### 아동정보 테스트
```
□ 아동 추가 (모든 필드 입력)
□ 첨부파일 업로드 (이미지 선택)
□ 카드 수정 클릭 → 수정 가능 상태
□ 저장 클릭 → 데이터 저장
□ 삭제 클릭 → 아동 삭제
□ 브라우저 새로고침 후 데이터 유지
```

### 스케줄 테스트 (필요시)
```
□ 스케줄 페이지 열기
□ 요일 선택 (월, 수, 금 등)
□ 시간대 선택 (오전/오후)
□ 아동 선택
□ 저장 클릭 → 시간표에 표시
□ 중복 선택 수정
□ 완성된 시간표 화면 확인
```
