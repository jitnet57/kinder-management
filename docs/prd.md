## 2. 핵심 기능 정의

### 2.1 모듈 1: 스케줄 관리 (Schedule)

#### 요구사항
- **주간 시간표**: 월-토, 오전/오후 구분, 한 화면에 전체 표시
- **입력**: 아동명 + 세션명 + 시간대 + 요일 (중복 O)
- **색상**: 아동별 자동 할당 + 커스터마이징 가능
- **수정/삭제/저장**: 개별 + 다중 선택 일괄 작업
- **저장 형식**: 주간 단위 저장 (월~토)

#### 승인 기준
- [x] 시간표 충돌 감지 (同時間 같은 아동 불가)
- [x] 중복 선택 지원
- [x] 색상 직관성 (최소 8가지 색)

---

### 2.2 모듈 2: 아동정보

#### 요구사항
- **카드형 레이아웃**: 그리드 (반응형 2-4열)
- **필드**: 이름, 생년월일, 전화, 주소, 파일 첨부, 비고
- **액션**: 각 카드별 수정/저장/삭제
- **검색**: 아동명 검색 지원

#### 승인 기준
- [x] 파일 업로드 (5MB 이하, jpg/png)
- [x] 모바일 반응형 (1열로 축소)

---

### 2.3 모듈 3: 데이터 기록지

#### 요구사항
- **진입점**: 아동정보 탭 → 아동명 클릭
- **과제 카드**: Curriculum에서 선택한 과제 표시
- **일일 입력**: 날짜 선택 → 과제별 점수 입력
- **그래프**: 과제별 "그래프" 버튼 → 최근 7일 추세
- **비고**: 과제별 textarea (여러 줄)

#### 승인 기준
- [x] 동일 과제 여러 날 입력 가능
- [x] 그래프 7일 자동 수집
- [x] 수정/삭제/저장 per 과제

---

### 2.4 모듈 4: 완료목록

#### 요구사항
- **자동 집계**: SessionLog에서 COMPLETED 상태만 추출
- **필터**: 아동별, 날짜별, 과제별
- **표시**: 완료율 + 최근 활동

#### 승인 기준
- [x] 실시간 업데이트

---

### 2.5 모듈 5: 커리큘럼

#### 요구사항
- **계층**: 발달영역 → LTO → STO → 비고
- **편집**: 추가/수정/삭제 (관리자만)
- **연동**: SessionLog의 드롭다운으로 노출
- **정렬**: 수동 정렬 또는 자동 정렬

#### 승인 기준
- [x] 50개 이상 커리큘럼 지원
- [x] 카테고리 추가 가능

## 3. 사용자 여정 (User Journey)

### 3.1 실무자 일일 흐름

```
[로그인]
  ↓
[대시보드 → 아동 목록 조회]
  ↓
[아동 선택 → 데이터 기록지 열기]
  ↓
[과제 선택 (Curriculum 드롭다운)]
  ↓
[오늘 날짜 입력]
  ↓
[점수 입력 + 비고 작성]
  ↓
[저장]
  ↓
[그래프 확인 (선택사항)]
  ↓
[완료]
```

```
[로그인]
  ↓
[스케줄 탭]
  ↓
[주간 시간표 조회]
  ↓
[아동 추가/편집]
  ↓
[충돌 확인]
  ↓
[저장]
  ↓
[리포팅 (완료목록 확인)]
```

## 4. 기술 스택 (Technology Stack)

### 4.1 프론트엔드
- **프레임워크**: React 18 + TypeScript
- **스타일**: Tailwind CSS + HeadlessUI
- **상태 관리**: Zustand 또는 TanStack Query
- **차트**: Recharts (경량, React 친화)
- **파형**: Next.js 13+ (선택사항, SSR/SSG)

### 4.2 백엔드
- **런타임**: Node.js 18+
- **프레임워크**: Hono (가볍고 빠름) 또는 Express
- **ORM**: Prisma (타입 안전성)
- **인증**: JWT + httpOnly cookies
- **유효성 검사**: Zod (스키마 검증)

### 4.3 데이터베이스
- **DBMS**: PostgreSQL 14+
- **호스팅**: Supabase (또는 자체 호스팅)
- **마이그레이션**: Prisma migrations

### 4.4 배포
- **프론트**: Vercel (또는 Netlify)
- **백엔드**: Fly.io (또는 Railway)
- **정적 파일**: S3-compatible (R2, Minio)
- **모니터링**: Sentry (에러 추적)
