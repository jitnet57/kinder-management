## 3. 모듈별 화면 설계

### 3.1 스케줄 관리 (Schedule Tab)

#### 레이아웃
```
┌─ Schedule Management ────────────────────┐
│                                          │
│ [주간 선택] [이전주] [이번주] [다음주]    │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━     │
│                                          │
│  시간    │ 월  │ 화  │ 수  │ 목  │ 금  │ 토 │
│  ───────┼─────┼─────┼─────┼─────┼─────┼─── │
│ 오전8-10│[A]  │     │[B]  │     │     │[A] │
│  ───────┼─────┼─────┼─────┼─────┼─────┼─── │
│ 오전10-12│[B] │[A] │     │[C]  │[B]  │    │
│  ───────┼─────┼─────┼─────┼─────┼─────┼─── │
│ 오후2-4 │[C] │[B] │[A]  │[A]  │[C]  │[B] │
│  ───────┼─────┼─────┼─────┼─────┼─────┼─── │
│ 오후4-6 │     │[A] │     │[B]  │[A]  │    │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━     │
│                                          │
│ [+ 새 일정 추가]                         │
│                                          │
│ ─────────────────────────────────────  │
│ 범례: [A] = 아동A (색상), [B] = 아동B   │
│      각 항목 클릭 → 수정/삭제 모달      │
│                                          │
└──────────────────────────────────────────┘
```

#### 상호작용
- **아동 추가**: "오후2-4" 셀 클릭 → 모달 open
  - 입력: 아동명 (드롭다운) | 세션명 (텍스트) | 시간 (시작~종료)
  - 버튼: [저장] [취소]
  
- **수정/삭제**: 셀의 [A] 클릭 → 팝업 메뉴
  - 옵션: [수정] [삭제] [닫기]

- **다중 선택**: Ctrl+Click 또는 체크박스
  - "선택 3개" → [일괄 삭제] [일괄 시간 변경] 버튼 활성화

#### 색상 코딩
- 각 [A], [B], [C]는 아동의 할당 색상 (위 1.1 참조)
- 호버: Opacity 0.8, Shadow 강조
