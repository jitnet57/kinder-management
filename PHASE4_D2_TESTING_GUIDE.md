## 🚀 테스트 환경 준비

### 사전 요구사항
```bash
# Node.js v18+ 설치 확인
node --version  # v18+

# 프로젝트 디렉토리
cd /e/kinder-management/frontend

# 의존성 설치
npm install

# 개발 서버 시작
npm run dev
```

### 접근 URL
```
http://localhost:5187

```

## 🚨 문제 해결 (Troubleshooting)

### 로그인 실패
```
증상: "이메일 또는 비밀번호가 올바르지 않습니다"
해결:
[ ] 이메일 정확히 입력 확인
    - Admin: admin@akms.com (소문자)
    - Therapist: therapist@akms.com
    - Parent: parent@akms.com
[ ] 비밀번호 확인
    - Admin: admin123
    - Therapist: therapist123
    - Parent: parent123
[ ] Caps Lock 확인
[ ] 브라우저 캐시 비우기 (Ctrl+Shift+Delete)
```

### 페이지가 로드 안 됨
```
증상: 흰 페이지 또는 로딩 중 상태
해결:
[ ] F12 → Console에서 에러 메시지 확인
[ ] 네트워크 탭에서 실패한 요청 확인
[ ] 브라우저 새로고침 (Ctrl+R 또는 F5)
[ ] 개발 서버 상태 확인 (npm run dev 실행 중인지)
[ ] localhost:5187 대신 localhost:5173 확인 (기본 포트)
```

### 메뉴가 표시 안 됨
```
증상: 메뉴 개수가 맞지 않음
해결:
[ ] 로그인한 역할 확인 (헤더의 이름과 이모지)
[ ] 페이지 새로고침
[ ] 브라우저 개발자 도구에서 localStorage 확인
    - 우클릭 → 검사 → Application → LocalStorage
    - akms_user 객체의 role 확인
```

### 콘솔 에러
```
증상: 빨간 에러 메시지
해결:
[ ] 에러 메시지 전체 기록
[ ] 에러 발생 시점 기록 (언제 발생했는지)
[ ] 재현 단계 기록
[ ] GitHub Issue로 보고
```
