## 📊 Executive Summary (요약)

| 항목 | myABAData | AKMS | 평가 |
|------|-----------|------|------|
| 커리큘럼 규모 | 14 도메인 × 240 LTO × 1,262 STO | 14 도메인 × 252 LTO × 1,008 STO | AKMS가 LTO 多 |
| 보안 수준 | 기본 인증 | **2단계 승인 + 디바이스 제한** | **AKMS 우위** |
| 데이터 기록 방식 | 빈도/지속시간/구간기록 | **ABC 데이터 기반** | **AKMS 우위** |
| 구독 모델 | 3단계 (개인/센터/멀티) | 없음 (엔터프라이즈) | myABAData 우위 |
| 실시간 시각화 | ✅ 있음 | ✅ 있음 (Recharts) | 동일 |
| 내보내기 | PDF/인쇄 | **Excel/Word** | **AKMS 우위** |
| 맞춤형 커리큘럼 | ✅ 추가 가능 | ⏳ 향후 계획 | myABAData 우위 |
| 모바일 | ✅ 반응형 | ✅ Tailwind 반응형 | 동일 |
| 배포 | Vercel (프로덕션) | **Cloudflare Pages** | 동일 |

## 🎯 커리큘럼 구조 비교

### myABAData
```
14 도메인
├─ 240 장기목표 (LTO)
└─ 1,262 단기목표 (STO)

특징:
• STO가 더 많음 (1,262개)
• 기본 커리큘럼 제공
• 사용자 정의 커리큘럼 추가 가능 ⭐
```

### AKMS
```
14 도메인
├─ 252 장기목표 (LTO) ⭐ More than myABAData
└─ 1,008 단기목표 (STO)
   ├─ 4개 STO per LTO
   └─ 각각: 보조 있이 → 부분적 독립 → 독립 → 일반화

특징:
• 더 세분화된 LTO 구조
• 체계적인 STO 진행 단계
• 교수 팁 8가지 포함 ⭐
• 자동 생성된 데이터
```

### 승자: **AKMS**
- LTO 수가 더 많음 (252 vs 240)
- 각 LTO마다 체계적인 4단계 STO
- 교수 팁 자동 생성

## 🔐 보안 & 승인 시스템

### myABAData
```
사용자 인증:
├─ 기본 로그인 (이메일/비밀번호)
├─ 기본 역할 분류 (therapist/admin/parent)
└─ 세션 관리

특징:
• 간단하고 빠른 인증
• 구독 기반 권한 관리
• 다중 직원 센터 지원
```

### AKMS
```
2단계 승인 시스템:
├─ 1단계: 관리자(시스템) 승인 ⭐⭐⭐
│   ├─ 디바이스 정보 검증
│   ├─ IP 주소 확인
│   └─ 사용자 정보 검증
├─ 2단계: 개발자(최종) 승인 ⭐⭐⭐
│   ├─ 비즈니스 규칙 확인
│   └─ 최종 승인
└─ 디바이스 제한
    ├─ 브라우저 핑거프린팅
    ├─ IP 주소 추적
    └─ 승인된 기기만 접근

특징:
• 이중 검증 (승인 권한 분산)
• 디바이스 레벨 보안
• 접근 로그 기록
• 거부 후 완전 차단
```

### 승자: **AKMS** 🏆
- 2단계 승인으로 보안 강화
- 디바이스 제한으로 물리적 보안
- 개발자(사용자) 최종 승인으로 신뢰도 ↑

### myABAData
```
측정 방식:
├─ 빈도 (Frequency)
│   └─ 행동 발생 횟수
├─ 지속시간 (Duration)
│   └─ 행동 지속 시간
├─ 구간기록 (Interval Recording)
│   └─ 일정 간격 단위 기록
└─ 데일리시트
    └─ 각 세션마다 시도별 기록

```

### AKMS
```
ABC 데이터 기반 (ABA 표준):
├─ A (Antecedent): 선행사건
│   └─ "어떤 상황에서"
├─ B (Behavior): 행동
│   └─ "아동이 어떻게 반응했는가"
└─ C (Consequence): 결과
    └─ "그 후 어떻게 되었는가"

구조:
├─ STO별 진행 상태
│  ├─ 1단계: 보조 있이 시도
│  ├─ 2단계: 부분적 독립
│  ├─ 3단계: 독립적 수행
│  └─ 4단계: 일반화
├─ 세션 기록
│  ├─ 시작/종료 시간
│  ├─ 점수 (0-100)
│  └─ 특이 사항 노트
└─ 교수 팁 자동 제공
    ├─ 보조 계층
    ├─ 강화 방법
    ├─ 오류 수정
    ├─ 일반화 전략
    └─ 기타 7가지

특징:
• 행동분석 심화 기록
• 개입 효과 분석 가능
• 교수 방법 지원
• 개입 최적화 가능
```

### 승자: **AKMS** 🏆
- ABC 기반으로 행동분석 심화
- 단계별 진행 추적으로 세밀한 제어
- 교수 팁으로 치료사 가이드
