# 📊 Dashboard + Reports 다양한 그래프 기능

**목표**: 대시보드와 보고서에서 다양한 그래프를 선택하여 표시  
**상태**: ✅ 구현 완료  
**날짜**: 2026-04-26

---

## 🎯 구현된 기능

### 1️⃣ Dashboard - 동적 그래프 선택

#### 위치
```
대시보드 페이지 → 하단 "📈 성과 분석" 섹션
```

#### 기능
```
✅ 8가지 차트 타입 선택
   ├─ 📈 추세 그래프 (Line Chart)
   ├─ 📊 막대 그래프 (Bar Chart)
   ├─ 🥧 원형 그래프 (Pie Chart)
   ├─ 🔥 히트맵 (Heatmap)
   ├─ 📐 히스토그램 (Histogram)
   ├─ 📚 스택 바 (Stacked Bar)
   ├─ 💫 산점도 (Scatter)
   └─ 🌊 영역 그래프 (Area)

✅ 시간 범위 선택
   ├─ 📅 이번주 (7일)
   ├─ 📅 이번달 (30일)
   ├─ 📅 분기 (90일)
   └─ 📅 연간 (365일)

✅ 그룹화 방식 선택
   ├─ 일별
   ├─ 주별
   ├─ 월별
   ├─ 아동별
   └─ 발달영역별

✅ 실시간 통계 표시
   ├─ 평균 점수
   ├─ 추세 (↗️ ↘️ →)
   └─ 향상도 (%)
```

#### 사용 흐름

```
Dashboard 진입
    ↓
"📈 성과 분석" 섹션 보기
    ├─ 8개 차트 버튼
    ├─ 시간 범위 드롭다운
    ├─ 그룹화 방식 드롭다운
    └─ 선택 시 자동 그래프 변경 (API 호출)
    ↓
원하는 그래프 보기
    ├─ 추세 보기
    ├─ 비교하기
    ├─ 통계 확인
    └─ "상세 보고서 보기" → Reports 페이지로 이동
```

---

### 2️⃣ Reports - 전체 분석 대시보드

#### 위치
```
메인 메뉴 → "📊 보고서" (신규 추가)
```

#### 기능
```
✅ 위와 동일한 8가지 차트 타입
✅ 위와 동일한 시간 범위/그룹화
✅ 추가 필터:
   ├─ 특정 아동 선택
   ├─ 발달영역 필터
   └─ 커리큘럼 필터

✅ 상세 기능:
   ├─ 통계 박스 (평균, 추세, 향상도)
   ├─ 데이터 테이블 표시 (10행 + 더보기)
   ├─ JSON 내보내기
   ├─ PNG 내보내기 (준비 중)
   └─ PDF 내보내기 (준비 중)

✅ 설정 자동 저장
   └─ 마지막 선택한 차트/범위 기억
```

## 📊 각 차트의 특징

### 📈 추세 그래프 (Line Chart)
```
용도: 시간대별 성과 변화 추적
예시:
  4/20: 75점
  4/21: 78점
  4/22: 82점 ← 상향 추세
  4/23: 80점
  4/24: 85점
  4/25: 88점
  4/26: 90점

장점:
  ✅ 시간 흐름 명확히 보임
  ✅ 추세 분석 용이
  ✅ 개선 추적 가능
```

### 📊 막대 그래프 (Bar Chart)
```
용도: 아동별 성과 비교
예시:
  민준: 88점
  소영: 82점
  지호: 79점
  영희: 85점

장점:
  ✅ 순위 비교 명확
  ✅ 개별 성과 한눈에
  ✅ 강점/약점 파악
```

### 🥧 원형 그래프 (Pie Chart)
```
용도: 커리큘럼 완료율 (완료/진행중/대기)
예시:
  완료: 85% (120개)
  진행중: 13% (18개)
  대기: 2% (3개)

장점:
  ✅ 비율 한눈에
  ✅ 전체 현황 파악
  ✅ 완료도 시각화
```

### 🌊 영역 그래프 (Area Chart)
```
용도: 발달영역별 누적 성과
예시:
  운동능력: 70 → 82 (↗️)
  인지능력: 75 → 90 (↗️)
  사회성: 72 → 88 (↗️)

장점:
  ✅ 여러 지표 동시 추적
  ✅ 누적 비율 표시
  ✅ 균형잡힌 성장 확인
```

### 💫 산점도 (Scatter Chart)
```
용도: 시간대별 점수 분포
예시:
  아침 (8-10시): 중간~높음
  낮 (12-14시): 낮음~중간
  오후 (14-18시): 높음
  저녁 (18-20시): 중간

장점:
  ✅ 시간대별 성과 분석
  ✅ 아동별 패턴 파악
  ✅ 최적 시간 찾기
```

### 📐 히스토그램 (Histogram)
```
용도: 점수 분포 분석
예시:
  60-70점: 5명
  70-80점: 15명
  80-90점: 25명
  90-100점: 12명

장점:
  ✅ 점수 분포 시각화
  ✅ 정상 분포 확인
  ✅ 이상치 감지
```

### 📚 스택 바 (Stacked Bar)
```
용도: 발달영역별 누적 성과 비교
예시: 아동별로 보기
  민준: 운동(75) + 인지(82) + 사회(78) = 235
  소영: 운동(80) + 인지(85) + 사회(81) = 246
  지호: 운동(78) + 인지(80) + 사회(88) = 246

장점:
  ✅ 영역별 기여도 명확
  ✅ 전체 vs 부분 비교
  ✅ 발달 불균형 파악
```

### 🔥 히트맵 (Heatmap)
```
용도: 요일/시간대별 활동 강도
예시:
  월요일 오전: 활발
  수요일 오후: 매우 활발
  금요일 저녁: 보통
  
색상:
  🔴 빨강: 매우 높음
  🟡 노랑: 보통
  🟢 초록: 낮음
```

## 🛠️ 기술 구현

### API 엔드포인트

```
POST /api/reports/generate
├─ 요청:
│  ├─ chartType: string (8가지 중 선택)
│  ├─ timeRange: string (7d, 30d, 90d, 365d)
│  ├─ groupBy: string (daily, weekly, monthly, child, domain)
│  ├─ childId?: number (선택)
│  ├─ domainId?: string (선택)
│  └─ includeStats: boolean
│
└─ 응답:
   ├─ status: "success"
   ├─ chart:
   │  ├─ chartType: string
   │  ├─ title: string
   │  ├─ data: any[]
   │  └─ stats: {average, trend, improvement}
   └─ timestamp: string

GET /api/reports/options
├─ 응답: 모든 차트/범위/그룹 옵션 나열

POST /api/reports/export
├─ 요청: format (json, png, pdf), chart 데이터
└─ 응답: 파일 다운로드

GET /api/reports/quick
└─ 응답: Dashboard용 미리보기 (2개 차트)
```

pages/Dashboard.tsx

## 📈 사용자 시나리오

### Scenario 1: 주간 성과 확인 (3분)

```
1️⃣ Dashboard 진입
   └─ 통계 카드: 12명, 48세션, 85% 완료율 확인

2️⃣ "성과 분석" 섹션에서 그래프 확인
   ├─ 기본: 추세 그래프 (자동 표시)
   └─ 보기: 이번주 7일 데이터

3️⃣ 통계 확인
   ├─ 평균: 83.3점
   ├─ 추세: ↗️ 상향
   └─ 향상도: +15%

4️⃣ 막대 그래프로 변경
   └─ 아동별 비교 (4명 점수 비교)

5️⃣ 상세 보고서로 이동
   └─ Reports 페이지 → 더 깊은 분석
```

### Scenario 2: 발달영역별 분석 (5분)

```
1️⃣ Dashboard에서 그룹화 방식 변경
   ├─ 기본: "일별" → "발달영역별"
   └─ 차트 자동 갱신

2️⃣ 영역 그래프 선택
   └─ 운동, 인지, 사회 3개 영역의 성과 비교

3️⃣ 불균형 발견
   ├─ 운동능력: 78점
   ├─ 인지능력: 85점 (높음)
   └─ 사회성: 88점 (최고)
   
   → 운동능력 강화 프로그램 추가 필요

4️⃣ Reports에서 상세 필터
   ├─ 운동능력만 선택
   ├─ 지난 30일 보기
   └─ 개별 아동별로 분석
```

### Scenario 3: 아동별 비교 (10분)

```
1️⃣ Dashboard → Reports 이동

2️⃣ 아동별 필터 적용
   ├─ 민준, 소영, 지호 선택
   └─ 자동 필터링

3️⃣ 막대 그래프로 비교
   ├─ 민준: 88점 (최고)
   ├─ 소영: 82점
   └─ 지호: 79점

4️⃣ 개인별 추세 확인
   ├─ 각 아동 추세 그래프 생성
   └─ 향상도 계산

5️⃣ 보고서 내보내기
   └─ JSON으로 다운로드 (추후 PDF/PNG)
```
