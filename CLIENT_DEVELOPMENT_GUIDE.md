**버전**: 1.0

## 🎯 프로젝트 개요

### 1️⃣ 프로젝트 목표

AKMS는 **장애 아동의 행동 분석 및 치료 관리를 위한 통합 디지털 플랫폼**입니다.

#### 주요 기능
- 📊 **ABC 분석 시스템** - 행동 데이터 기록 및 분석
- 📚 **커리큘럼 관리** - 늘품 기반 개별화 교육 계획 수립
- 📈 **진행 추적** - 아동별 학습 진도 및 통계
- 💬 **협력 플랫폼** - 치료사와 부모 간 실시간 소통
- 📱 **모바일 접근** - 언제 어디서나 정보 확인

- 아동별 진도 추적
- 개별 대시보드

✅ **대시보드**
- 실시간 통계
- 진도율 표시
- 최근 활동 로그

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 시스템은 얼마나 안전한가요?
**A**: 
- 클라우드 보안 표준(HTTPS, 데이터 암호화) 준수
- 정기적 보안 점검 예정
- 데이터 자동 백업 (일 1회)
- 자세한 내용은 보안 정책 문서 참고

### Q2. 얼마나 많은 데이터를 처리할 수 있나요?
**A**:
- 초기: 100-200명 아동 데이터 처리 가능
- 확장: 클라우드 리소스 증설로 무제한 확장 가능
- 성능: 응답 시간 < 2초 보장

### Q3. 부모들도 사용할 수 있나요?
**A**:
- Phase 3: 기본 기능만 (현재)
- Phase 6 (5월 중): 부모 전용 대시보드 추가 예정
- 부모 자동 로그인 기능도 구현

### Q4. 기존 데이터를 옮길 수 있나요?
**A**:
- CSV/Excel 파일 일괄 import 기능 예정
- 수동 마이그레이션도 가능
- Phase 5b에서 데이터 통합

### Q5. 모바일에서도 사용 가능한가요?
**A**:
- 현재: 모바일 웹브라우저 지원
- 5월: 반응형 디자인 개선
- 6월: 네이티브 모바일 앱 검토

### Q6. 비용이 추가로 드나요?
**A**:
- 호스팅: 무료 (Cloudflare Pages)
- 도메인: 무료 (Cloudflare 제공)
- 이후 확장 시만 유료 리소스 추가 필요
