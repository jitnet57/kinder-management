1. 전체 구조

총 60개의 새로운 LTO 생성:

  - Domain Writing (쓰기): 15개 LTO
  - Domain Math (수학): 15개 LTO

각 LTO의 구조:
  - ID: 도메인_ltoXX (예: domain_visual_lto01)
  - Order: 1-15 (발달 순서대로 배열)
  - Name: 목표 기술명
  - Goal: 최종 목표 설명
  - STO Count: 4개 (Support To Independence)
  - STOs: 보조 있이 → 부분적 독립 → 독립적 수행 → 일반화 (4단계)
  - TeachingTips: 8개 항목 (prompt_hierarchy, reinforcement, error_correction 등)

2. Domain Visual Perception (시각 지각) - 15개 LTO
================================================================================

LTO01: 색상 구분하기 (빨강, 파랑, 노랑, 초록)
LTO02: 형태 인식하기 (원, 삼각형, 네모)
LTO03: 패턴 인식하기 (2-3항목 반복 패턴)
LTO04: 크기 비교하기 (크고, 작고, 중간 크기)
LTO05: 위치 개념 이해하기 (위/아래, 앞/뒤, 안/밖, 옆)
LTO06: 도형의 특징 이해하기 (변, 꼭짓점, 곡선)
LTO07: 그림 일치하기 (동일한 그림 2개 찾기)
LTO08: 그림 차이 찾기 (거의 동일한 2개 그림에서 차이)
LTO09: 색칠하기 (정해진 영역 내 색칠)
LTO10: 점 연결하기 (번호 순서 따라 점 연결)
LTO11: 같은 것 찾기 (같은 종류의 물건/그림 찾기)
LTO12: 서로 다른 것 찾기 (같은 종류 중 다른 것 1개 찾기)
LTO13: 숨겨진 그림 찾기 (복잡한 배경 속 특정 그림 찾기)
LTO14: 시각적 순서 이해하기 (3-4단계 순차 그림 배열)
LTO15: 공간 개념 이해하기 (자신의 신체와 물체의 공간 관계)

발달 순서: 기초 색상/형태 → 시각 조직화 → 정보 처리 → 공간 추론

3. Domain Writing (쓰기) - 15개 LTO
================================================================================

LTO01: 선 긋기 (수평선, 수직선, 대각선)
LTO02: 원 그리기 (크레용/연필로 원 반복 그리기)
LTO03: 삐침 따라 그리기 (구불구불한 선/곡선)
LTO04: 점선 따라 연결하기 (점선 형태 선으로 완성)
LTO05: 문자 모양 이해하기 (기본 문자 인식)
LTO06: 대문자 쓰기 (자음 기본 문자 필기)
LTO07: 소문자 쓰기 (모음, 작은 자음 필기)
LTO08: 숫자 쓰기 (숫자 1-10 필기)
LTO09: 이름 쓰기 (자신의 이름 2-4글자 필기)
LTO10: 단어 쓰기 (3-4글자 일상 단어)
LTO11: 짧은 문장 쓰기 (2-3어절 문장)
LTO12: 긴 문장 쓰기 (4-5어절 이상 복합 문장)
LTO13: 자신의 생각 표현하기 (질문에 대한 답을 문장으로)
LTO14: 편지 쓰기 (인사→내용→맺음말 기본 구조)
LTO15: 창작 글쓰기 (시각 자료 없이 자신의 상상으로 창작)

발달 순서: 기초 필기 기술 → 문자/숫자 → 단어 → 문장 → 자기 표현 → 창작

4. Domain Math (수학) - 15개 LTO
================================================================================

LTO01: 1-5 숫자 인식 (숫자 1-5를 음성으로 인식하고 수량 매칭)
LTO02: 6-10 숫자 인식 (숫자 6-10 인식 및 수량 매칭)
LTO03: 11-20 숫자 인식 (숫자 11-20 인식)
LTO04: 수량 세기 (주어진 물건을 일대일 대응으로 세기)
LTO05: 일대일 대응 (두 그룹 물건 짝짓기)
LTO06: 수의 크기 비교 (두 수량 비교)
LTO07: 더하기 (1+1~5+5)
LTO08: 빼기 (5-1~10-5)
LTO09: 돈 세기 (동전/지폐 가치 이해 및 합계 계산)
LTO10: 시간 읽기 (아날로그 시계 정각과 반 읽기)
LTO11: 길이 측정 (비표준 단위로 길이 측정)
LTO12: 무게 비교 (두 물체 무게 비교)
LTO13: 도형의 성질 (변, 꼭짓점 세기 및 분류)
LTO14: 규칙 찾기 (수열/도형 배열 규칙 발견)
LTO15: 문제 해결 (간단한 수학 상황에서 덧셈/뺄셈으로 해결)

발달 순서: 수 인식 → 계산 → 측정 → 응용 능력

5. Domain Reading (읽기) - 15개 LTO
================================================================================

LTO01: 글자 인식 (기본 문자 시각적 인식)
LTO02: 낱말 읽기 (1-2음절) (엄마, 아빠, 손, 발)
LTO03: 낱말 읽기 (3음절 이상) (자동차, 아이스크림)
LTO04: 짧은 문장 읽기 (2-3어절 간단한 문장)
LTO05: 긴 문장 읽기 (4-5어절 이상 복합 문장)
LTO06: 이야기 읽기 (3-5문장의 짧은 이야기)
LTO07: 이해력 (주제 파악) (읽은 이야기의 주제)
LTO08: 이해력 (세부 내용) (인물, 장소, 사건 등 세부 정보)
LTO09: 이해력 (인과관계) (원인-결과 이해)
LTO10: 이야기 순서 정렬 (뒤섞인 문장/사진을 올바른 순서로)
LTO11: 예측 능력 (다음에 무엇이 일어날지 예측)
LTO12: 느낌 표현 (읽은 내용에서 감정 표현)
LTO13: 정보 찾기 (텍스트에서 특정 정보 찾기)
LTO14: 요약하기 (읽은 이야기를 짧게 다시 말하기)
LTO15: 비판적 읽기 (읽은 내용에 대해 질문/의견/평가)

발달 순서: 글자 인식 → 단어/문장 읽기 → 이해 → 사고 능력

================================================================================
6. 각 LTO의 핵심 특징
================================================================================

A. 4단계 STO 구조 (ABA 기반)
   Stage 1: 보조 있이 시도 (Full support)
   Stage 2: 부분적 독립 (Partial independence with support)
   Stage 3: 독립적 수행 (Independent performance)
   Stage 4: 일반화 (Generalization across contexts)

B. TeachingTips 8가지 항목
   1. Prompt Hierarchy: 단계적 보조 전략
   2. Reinforcement: 강화 방법
   3. Error Correction: 오류 교정 방법
   4. Generalization: 일반화 전략
   5. Data Collection: 자료 수집 방법
   6. Prerequisite: 선행 기술
   7. Motivation: 동기 유발 방법
   8. Family Involvement: 가정 연계 활동
