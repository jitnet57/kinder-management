## 📋 Phase 5 주요 미션

### 1️⃣ WiFi 기반 자동 동기화 (자동화)

#### 현재 상태
```
수동: USB로 백업 파일 이동
        ↓
복원 API 호출
        ↓
데이터 동기화
```

#### Phase 5 목표
```
자동: 같은 WiFi 네트워크 감지
        ↓
백그라운드 동기화 (매시간)
        ↓
양방향 데이터 병합
        ↓
충돌 해결 (최신 데이터 우선)
```

#### 구현 계획

**Step 1: 네트워크 감지 (1주)**
```typescript
// frontend/src/utils/networkSync.ts
class WiFiSync {
  detectNetwork(): Promise<{ ssid, strength }>;
  isInOfficeNetwork(): boolean;
  onNetworkChange(): Observable<NetworkStatus>;
}
```

**Step 2: 자동 동기화 엔진 (2주)**
```typescript
// backend/src/services/AutoSyncService.ts
class AutoSyncService {
  async scheduleSync(interval: number);  // 매시간
  async mergeDatabases();                 // 양방향 병합
  async resolveConflicts();               // 충돌 해결
}
```

**Step 3: UI 표시 (1주)**
```typescript
// frontend/src/components/SyncStatus.tsx
<SyncStatus
  status="syncing"     // idle, syncing, error
  lastSync="10분 전"
  conflicts={0}
/>
```

### 3️⃣ AI 기반 분석 (지능화)

#### 기능 1: 자동 보고서 생성

```
세션 데이터 (일일 입력)
        ↓
LLM 분석 (Claude API)
        ↓
insights 추출
        ├─ "지난주 사교성 20% 향상"
        ├─ "언어 기술 평탄"
        └─ "집중력이 장시간 유지됨"
        ↓
📄 자동 보고서 (PDF)
```

**구현** (3주):
```typescript
// backend/src/services/InsightService.ts
async generateInsights(childId: number): Promise<Insight[]> {
  const logs = await getPreviousLogs(childId, '7d');
  const insights = await claude.generateInsights(logs);
  return insights;
}
```

#### 기능 2: 예측 분석

```
지난 3개월 데이터
        ↓
트렌드 분석
        ├─ 개선: ↗️
        ├─ 침체: ↘️
        └─ 변동: ↔️
        ↓
다음 주 목표 제안
└─ "언어 기술 강화 권장"
```

**구현** (2주):
```typescript
async predictNextGoal(childId: number): Promise<Recommendation> {
  const trend = calculateTrend(logs);
  const recommendation = await generateRecommendation(trend);
  return recommendation;
}
```

#### 기능 3: 비교 분석

```
아동 A vs 아동 B
        ↓
강점/약점 비교
├─ 아동 A: 사교성 강함
├─ 아동 B: 자기조절력 강함
└─ 모두: 집중력 개선 필요
        ↓
개별 맞춤 제안
```
