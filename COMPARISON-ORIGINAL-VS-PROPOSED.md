# 기존 요구사항 vs 제안 비교표

**작성**: 2026-04-26  
**목적**: 벤치마킹 기반 전략과 원래 요구사항의 차이점 시각화

---

## 1️⃣ 핵심 기능 비교

| 기능 | 기존 요구사항 | 제안 사항 | 개선점 | 영향도 |
|------|-------------|---------|-------|--------|
| **스케줄 관리** | 월-토 시간표 (한 화면) | 월-토 시간표 + 월간 뷰 + 충돌 경고 | 유연성 증가, 스케줄 충돌 사전 예방 | ⭐⭐⭐ |
| **아동 정보** | 카드형 (이름, 생년월일, 전화, 주소, 파일) | 카드형 + 성장 지표 (발달영역별 진행도) | 한눈에 성과 파악, 부모 보고서 용이 | ⭐⭐⭐ |
| **데이터 기록** | 일일 입력 (점수 + 비고) | 일일 입력 + 7/30일 추세 그래프 + 성장 분석 | 기존과 동일 (이미 구현됨) | ✅ |
| **완료 목록** | 기록지 → 자동 이동 | 필터 + 정렬 + 기간별 분석 추가 | 보고서 생성 시간 60% 단축 | ⭐⭐ |
| **커리큘럼** | 발달영역 → LTO → STO (드롭다운) | 트리 뷰 (확장/축소) + 진행도 시각화 | 계층 구조 더 직관적 | ⭐⭐ |

| **포커스 상태** | 없음 (마우스 중심) | **2px Teal 테두리** 추가 | ✅ 키보드 네비 지원 |
| **명도비** | 4.5:1 (텍스트) | 4.5:1 이상 (WCAG 2.1 AA) | ✅ 접근성 준수 |

## 3️⃣ 정보 아키텍처 비교

### 기존 (심플)
```
Side Navigation:
├─ 대시보드
├─ 스케줄
├─ 아동정보
├─ 데이터기록
├─ 완료목록
└─ 커리큘럼
```

### 제안 (계층적)
```
Top Bar: 로고 + 빠른 통계 (오늘 세션, 진행도) + 프로필

Side Navigation (계층적):
├─ 📊 대시보드
│  └─ Quick Actions (+ 새 아동, + 세션 기록, 보고서)
├─ 📅 스케줄
│  └─ 주간 뷰, 월간 뷰, 충돌 경고
├─ 👶 아동 관리
│  ├─ 목록 뷰 (빠른 검색)
│  └─ 상세 프로필 (파일, 진행도, 히스토리)
├─ 📊 성과 기록
│  ├─ 일일 기록 (빠른 입력)
│  └─ 분석 (7일, 30일, 맞춤)
├─ ✅ 완료 추적
│  └─ 필터 (기간, 아동, 상태)
└─ 🎓 커리큘럼
   ├─ 계층 뷰
   └─ 할당 및 진행도
```

| 비교 | 기존 | 제안 | 효과 |
|------|-----|------|------|
| **네비 깊이** | 1단계 (플랫) | 2-3단계 (계층적) | 인지 부하 감소 |
| **Quick Action** | 없음 | Top에 배치 | 자주 쓰는 기능 빠른 접근 |
| **통계 표시** | 대시보드만 | Top bar (항상 보임) | 실시간 현황 파악 |
| **아이콘** | 없음 | 이모지 추가 (👶 👶 📊) | 시각적 구분 |

## 4️⃣ 콘텐츠 전략 비교

| 항목 | 기존 | 제안 | 이점 |
|------|-----|------|------|
| **메시징** | 기능 중심 ("점수 기록", "세션 입력") | **결과 중심** ("아동이 성장한다", "새 기술 습득") | 부모 공감도 ↑ |
| **대상 사용자** | 임상가 중심 | **다중 이해관계자** (관리자/임상가/부모) | 접근성 ↑, 만족도 ↑ |
| **신뢰 신호** | 없음 | 대시보드에 추가 (HIPAA, 99.5% 가용성, 자동 백업) | 기관 신뢰도 ↑ |
| **다국어** | 한국어만 | **한국어 + 영어** (향후 확장) | 국제 시장 대비 |
| **접근성** | 기본 | **WCAG 2.1 AA 준수** (키보드, 스크린리더) | 법률 준수, 포용성 |

## 5️⃣ 사용자 흐름 비교

### 기존 (직선형)
```
로그인 → 대시보드 → 원하는 페이지 선택 → 작업 수행
```

### 제안 (점진적 참여 모델)
```
로그인
  ↓
[Step 1] 기관 정보 입력
  ↓
[Step 2] 첫 아동 추가
  ↓
[Step 3] 커리큘럼 할당
  ↓
[Step 4] 첫 세션 기록
  ↓
대시보드 (성장 대시보드 보이기)
```

| 비교 | 기존 | 제안 | 개선 |
|------|-----|------|------|
| **온보딩** | 없음 | 5단계 가이드 | 사용 난이도 ↓ |
| **첫 사용 경험** | 빈 화면 | 체크리스트 + 안내 | 신뢰감 ↑ |
| **인지 부하** | 한번에 모든 기능 | 순차적 학습 | 이탈률 ↓ |

## 6️⃣ 대시보드 비교

### 기존 (심플)
```
┌─────────────────────────────────┐
│ 4개 통계 카드 (숫자만)         │
│ - 등록 아동: 12                │
│ - 이번 주 세션: 48             │
│ - 완료율: 85%                  │
│ - 활성 커리큘럼: 24            │
│                               │
│ 최근 활동 로그 (텍스트)         │
│ 주간 스케줄 요약               │
└─────────────────────────────────┘
```

### 제안 (결과 + 신뢰 + 액션)
```
┌──────────────────────────────────────────────┐
│  KINDER ABA | 아동 ABA 관리 시스템            │
├──────────────────────────────────────────────┤
│ 📱 빠른 통계            🏥 기관 신뢰 (우측)  │
│  ├─ 등록: 12            ├─ ✓ HIPAA 준수    │
│  ├─ 진행도: 65% ↑       ├─ ✓ 암호화 저장   │
│  ├─ 총 세션: 240        ├─ ✓ 자동 백업    │
│  └─ 커리큘럼: 24        └─ ✓ 99.5% 가용성  │
│                                            │
│ 🏆 오늘 할일 (Pinned)                      │
│  ├─ [+ 세션 기록]                         │
│  ├─ [+ 스케줄]                           │
│  └─ [📊 보고서]                          │
│                                            │
│ 🎯 아동별 성장                            │
│  ├─ 아이린: 언어 발달 3개 목표 달성 ✨    │
│  ├─ 준호: 진행도 72% (↑ 68%)            │
│  └─ 유미: 6일 연속 기록                   │
└──────────────────────────────────────────────┘
```

| 항목 | 기존 | 제안 | 차이 |
|------|-----|------|------|
| **시각적 계층** | 4개 카드만 | 통계 + 신뢰 신호 + 액션 + 성과 | 정보 밀도 증가 |
| **신뢰도** | 없음 | 명시적 표시 | 부모/관리자 신뢰 ↑ |
| **빠른 작업** | 없음 | 3가지 버튼 고정 | 클릭 수 감소 |
| **성과 강조** | 없음 | 아동별 성장 하이라이트 | 동기부여 ↑ |

## 7️⃣ 접근성 비교

| 기능 | 기존 | 제안 | 상태 |
|------|-----|------|------|
| **WCAG 2.1 AA** | 미확인 | **표준 준수 목표** | ✅ 체크리스트 작성 |
| **키보드 네비** | 마우스 중심 | **전체 페이지 Tab 네비 가능** | ✅ 포커스 테두리 추가 |
| **스크린리더** | 기본 HTML | **aria-label, alt 텍스트 전체 추가** | ✅ 계획 중 |
| **명도비** | 확인 필요 | **4.5:1 이상 강제** | ✅ 모든 텍스트 검증 |
| **색상만으로 정보 표현** | 있음 | **아이콘 + 텍스트 병행** | ✅ 개선 |
| **오프라인 기능** | 없음 | **LocalStorage 캐싱** | 🔄 검토 중 |
