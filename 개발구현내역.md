# AKMS 시스템 개발 구현 내역

## 프로젝트 개요
**프로젝트명**: AKMS (ABA Child Management System)  
**목적**: 유아 ABA 데이터 관리 및 분석 시스템  
**기술스택**: React + TypeScript, Tailwind CSS, Recharts, Vite, Cloudflare Pages

---

## 완료된 주요 기능

### 1. 대시보드 (Dashboard)
- ✅ **통계 카드**: 등록된 아동, 이번 주 세션, 완료율, 활성 커리큘럼
- ✅ **호버 효과**: 통계 카드에서 색상 진화 (진한 색으로 변경)
- ✅ **상세보기 기능**: 모달로 각 통계의 상세 정보 표시
- ✅ **내보내기 기능**: JSON 형식으로 데이터 다운로드
- ✅ **동적 그래프**: 8가지 차트 타입 (추세, 막대, 원형, 산점도, 히트맵, 히스토그램, 스택바, 영역)
- ✅ **시간 범위 필터**: 7일, 30일, 90일, 365일 선택
- ✅ **그룹화 옵션**: 일별, 주별, 월별, 아동별, 발달영역별

### 2. 주간 스케줄 (Schedule)
- ✅ **월-토 시간표**: 오전 8-10, 10-12, 오후 2-4, 4-6 시간대
- ✅ **아동별 색상 구분**: 각 아동마다 고유한 파스텔 색상
- ✅ **세션 추가 기능**: 각 시간대의 + 버튼으로 새 세션 추가
- ✅ **세션 수정/삭제**: Edit, Delete 버튼으로 관리
- ✅ **호버 효과**: 아이콘 색상 변경, 텍스트 가시성 개선
- ✅ **세션 카드**: 아동 이름, 세션명, 시간 표시

### 3. 아동정보 (Children)
- ✅ **카드형 UI**: 모든 아동의 정보를 카드로 표시
- ✅ **아바타 생성**: SVG 기반 자동 아바타 생성 (초성 + 색상)
- ✅ **정보 입력**: 이름, 생년월일, 전화, 주소, 기타 정보 관리
- ✅ **상세보기**: 아동 카드 클릭 시 상세 정보 및 과제 표시
- ✅ **수정/삭제**: Edit, Delete 버튼으로 관리
- ✅ **검색 기능**: 아동 이름으로 빠른 검색

### 4. 데이터 기록 (SessionLog)
- ✅ **아동별 과제 관리**: 완료되지 않은 세션 과제 목록
- ✅ **커리큘럼 연동**: 발달영역-LTO-STO 드롭다운 선택
- ✅ **점수/시간/비고 입력**: 상세 세션 데이터 기록
- ✅ **그래프 버튼**: 각 과제의 성과 그래프 표시
- ✅ **완료 기능**: 과제 완료 시 완료목록으로 이동
- ✅ **호버 효과**: 아이콘 애니메이션 (확대, 회전)

### 5. 완료목록 (Completion)
- ✅ **완료된 과제 표시**: 완료 상태의 모든 과제 표시
- ✅ **필터링**: 아동, 시간 범위, 정렬 옵션
- ✅ **통계 표시**: 이번 주 완료수, 총 완료수, 평균 점수
- ✅ **체크마크 아이콘**: CheckCircle2 아이콘으로 완료 상태 표시
- ✅ **호버 텍스트 개선**: 명도 대비 향상으로 가독성 개선

### 6. 커리큘럼 (Curriculum)
- ✅ **3단계 계층**: 발달영역 → LTO → STO
- ✅ **CRUD 기능**: 각 레벨에서 항목 추가, 수정, 삭제
- ✅ **드롭다운**: 순차적 선택으로 사용자 경험 개선
- ✅ **아이콘 호버 효과**: Edit, Trash2, Plus 아이콘 색상 변경

### 7. 보고서 (Reports)
- ✅ **아동별 보고서**:
  - 아동 선택 인터페이스
  - 통계 카드 (총 세션, 평균 점수, 개선도)
  - 점수 추이 라인 그래프
  - 발달영역별 점수 막대 그래프
  - 상세 데이터 테이블
  
- ✅ **전체 통계 보고서**:
  - 전체 요약 (아동수, 총 세션, 평균 점수)
  - 아동별 성과 비교 그래프
  - 발달영역별 평균 점수
  - 점수 분포 원형 그래프
  - 아동별 & 발달영역별 상세 통계 테이블

- ✅ **데이터 내보내기**: JSON 형식으로 다운로드

### 8. 도움말 (Help)
- ✅ **시스템 개요**: AKMS 소개 및 기본 개념
- ✅ **기능별 설명서**: 모든 주요 기능에 대한 상세 설명
- ✅ **아코디언 UI**: 섹션별 펼쳐보기/접기
- ✅ **팁 & 활용법**: 사용자를 위한 꿀팁 제공
- ✅ **빠른 참조**: 주요 단축키 및 아이콘 설명

## 설계 및 아키텍처

### 상태 관리
- **Context API 활용**:
  - `CurriculumContext`: 커리큘럼 구조 및 세션 과제
  - `ScheduleContext`: 주간 스케줄 데이터
  - `CacheContext`: 로컬 캐시 관리

### 데이터 구조
```typescript
// ScheduleSession
interface ScheduleSession {
  id: string;
  dayOfWeek: number;
  timeSlotIndex: number;
  childId: string;
  childName: string;
  sessionName: string;
  startTime: number;
  endTime: number;
  color: string;
}

// SessionTask
interface SessionTask {
  id: string;
  childId: string;
  domainId: string;
  ltoId: string;
  stoId: string;
  date: string;
  startTime: string;
  endTime: string;
  score: number;
  notes: string;
  completed: boolean;
  completedAt?: string;
}
```

### 시각 디자인
- **파스텔톤 글래스 모르피즘**: 부드럽고 현대적인 UI
- **아동별 색상 체계**: 
  - 민준: #FFB6D9 (핑크)
  - 소영: #B4D7FF (파랑)
  - 지호: #C1FFD7 (민트)
  - 연서: #FFE4B5 (크림)

### 반응형 디자인
- Tailwind CSS의 breakpoint 활용
- Mobile, Tablet, Desktop 지원

## 업그레이드된 UI/UX 기능

### 1. 호버 효과
- ✅ **아이콘 호버**: 모든 아이콘이 호버시 색상 변경
- ✅ **카드 호버**: 카드가 위로 살짝 이동 (transform: translateY)
- ✅ **버튼 호버**: 배경색 변경 및 그림자 효과
- ✅ **그룹 호버**: 부모 요소 호버시 자식 요소 스타일 변경

### 2. 색상 반전 및 진화
- ✅ **대시보드 카드**: 호버시 더 진한 색상으로 변경
- ✅ **완료목록**: 호버시 텍스트 색상 진행 (text-gray-900)
- ✅ **아이콘 색상**: Edit(파랑), Delete(빨강), Plus(회전)

### 3. 가독성 개선
- ✅ **텍스트 그림자**: drop-shadow-sm으로 텍스트 강조
- ✅ **배경 투명도**: 호버시 배경 투명도 증가
- ✅ **명도 대비**: WCAG 기준을 고려한 색상 조합
